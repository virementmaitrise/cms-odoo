Virement Maitrisé

Acceptez les virements bancaires instantanés sécurisés. Les paiements sont traités via Société Générale. Support intégré pour e-commerce, factures et paiements par QR code.

NOUS CONTACTER

Fonctionnalités Principales

Ce module Odoo 18 offre :

  • Intégration E-commerce : Acceptez les virements bancaires instantanés lors du paiement sur votre site Odoo. Les clients sont redirigés en toute sécurité pour finaliser leur paiement via Société Générale.
  • Liens de Paiement sur Factures : Génération automatique de liens de paiement et de QR codes sur les factures. Aucun module supplémentaire nécessaire - le support des factures est intégré.
  • Paiements par QR Code : Les clients peuvent scanner les QR codes sur les factures ou devis pour payer instantanément depuis leur application bancaire mobile.
  • Rapprochement Automatique : Les paiements sont automatiquement associés aux commandes et factures via des notifications webhook.
  • Support des Remboursements : Effectuez des remboursements totaux ou partiels directement depuis Odoo.
  • Support Multi-sociétés : Configurez différents identifiants pour chaque société dans votre instance Odoo.
  • Modes Test & Production : Testez votre intégration en toute sécurité avant la mise en production.

Comment Ça Marche

Pour l'E-commerce :

1. Le client ajoute des produits au panier et procède au paiement
2. Le client sélectionne "Virement Maitrise" comme méthode de paiement
3. Le client est redirigé vers la page de paiement sécurisée de Société Générale
4. Le client sélectionne sa banque et autorise le virement
5. Le client est redirigé vers votre boutique avec confirmation du paiement

Pour les Factures :

1. Activez l'affichage du QR code/lien de paiement dans les paramètres Virement Maitrise
2. Créez et envoyez vos factures comme d'habitude dans Odoo
3. Les factures incluent automatiquement un lien de paiement et un QR code
4. Les clients cliquent sur le lien ou scannent le QR code pour payer instantanément
5. Le statut de la facture est automatiquement mis à jour lors de la réception du paiement

Tous les statuts de paiement sont automatiquement synchronisés via des webhooks sécurisés.

Configuration

ÉTAPE 1 : Installer le Module
Installez le module Virement Maitrise depuis le menu Applications Odoo. Aucun module supplémentaire n'est requis pour le support des factures.

ÉTAPE 2 : Configurer les Identifiants
Allez dans Paramètres → Fournisseurs de Paiement → Virement Maitrise

  • Saisissez votre ID d'Application PIS
  • Saisissez votre Secret d'Application PIS
  • Téléversez votre Fichier de Clé Privée PIS
  • Choisissez l'état Test ou Activé

Les identifiants sont fournis par Société Générale lors de l'activation du service.

ÉTAPE 3 : Configurer les Options d'Affichage
Choisissez comment les options de paiement apparaissent sur les factures :

  • Inclure lien/QR sur les factures : Afficher les liens de paiement et QR codes sur les factures PDF

ÉTAPE 4 : Tester Votre Intégration
En mode test, vous pouvez simuler des paiements sans virements bancaires réels. Utilisez l'environnement de test pour valider votre configuration avant la mise en production.
